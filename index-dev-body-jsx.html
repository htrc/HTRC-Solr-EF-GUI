Search all volume fields 		Help

Volume-level metadata searching is case-insensitive.

To enter a title-based search, simply enter the plain text words into the Volume Metadata search box. For phrased-based searching quote the words "like this"

To search all metadata fields, tick the Search all volume fields below the search box.

To search specific volume metadata fields use the field:term form, such as names_t:Austen or names_t:"Austen, Jane" for a phrase-based search.

Wildcard matching includes using '?' for a single character and '*' for multiple characters. For example 'canad?' and '*land'.

Page-level Text: Help

To enter a Part-of-Speech (POS) search, simply enter the plain text words into the Page-level Text search box. For phrased-based searching quote the words "like this"

Page-level metadata searching is case-insensitive.

To gather all the matching results from one volume together when searching, tick the Sort & Group by volume checkbox below the search box.

To search specific languages and for particular parts of speech within that language (where available) use the checkboxes below the Page-level Text search box to have them include in the search.

For languages with part-of-speech indexing a checkbox is provided to enable/disable that entire language in the search. There is also a Search all Languages checkbox provided as a convenient way to override individual selections, and have the query terms entered apply to all languages and part-of-speech tags.

Wildcard matching includes using '?' for a single character and '*' for multiple characters. For example 'canad?' and '*land'.
